# AI-Repository

Reusable AI workflows and Codex skills for international B2B business development.

## AIGBD Workflow Skill

The `skills/aigbd-workflow` package documents the stable local AIGBD workflow for Browser Use + DeepSeek + Chrome automation.

It covers:

- one-click launcher and health checks
- Google Search and Google Maps candidate pools
- qualification, website analysis, customer background and classification
- contact, sales, decision and email intelligence
- local draft email generation
- Windows troubleshooting and safety guardrails

The repository intentionally excludes API keys, `.env`, customer CSV exports, logs, cookies, and generated emails.
